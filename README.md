# 🚀 Sparx Science AI Extension

An AI-powered Microsoft Edge browser extension that helps solve Sparx Science questions using OpenAI's GPT models.

## ✨ Features

- **AI-Powered Answers**: Get instant solutions to science questions
- **Detailed Explanations**: Learn with step-by-step educational explanations
- **Multi-field Support**: Works with Physics, Chemistry, Biology, and all STEM subjects
- **Customizable Settings**: Choose between GPT-3.5 Turbo and GPT-4
- **Easy to Use**: Simple, intuitive interface
- **Copy Functionality**: Quickly copy answers to clipboard
